# 나린위키 프로젝트 : NARIN #

나린위키는 [그누보드](http://sir.co.kr) 기반의 [위키](http://ko.wikipedia.org/wiki/Wiki) 입니다. 그누보드 플러그인으로 개발 되었으며 간단한 설치 후 사용할 수 있습니다. 나린위키에 관하여 더 알고싶으신 분은 [나린위키 매뉴얼](http://narinwiki.org/manual)을 읽어보시기 바랍니다.
